# Template · Beat Sheet · 结构节拍表

> The stage-5 structure deliverable. Fill the beats of the chosen methodology
> (`modules/02-structure.md`), annotated with runtime budget and dual-track pacing.

## Generic beat row · 通用节拍行
Each beat carries five things:

```
Beat 节拍        | what happens (one or two lines)
Runtime 时长     | ~minutes / seconds
Plot 情节        | 松 / 中 / 紧
Emotion 情感     | 轻 / 中 / 重
Setup/Payoff     | plant here? pay off here? (cross-ref the setup register)
```

## Pacing wave · 节奏波浪图
Sketch both tracks beneath the sheet to *see* the rhythm:

```
Plot   :  ─╱╲────╱╲──╱╲╱╲╲──────╱╲╱╲╲
Emotion:  ────╱╲───────╱╲──╱╲╱╲────╲╱╲
```
Avoid: both tracks low for long · both high for long · either track monotonically
rising with no relief.
