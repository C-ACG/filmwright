# Template · Scene Card · 场景卡

> The Stage-6 contract between outline, screenplay, direction, and continuity.

```text
SCENE ID: SC001                   STATUS: proposed / approved / locked / stale
SLUGLINE: INT./EXT. LOCATION - TIME
SEQUENCE / EPISODE:
POV:

DRAMATIC ACTION
  Goal:
  Obstacle:
  Tactic:
  Turn:
  Exit value / newly possible action:

RHYTHM
  Runtime:
  Plot: 松 / 中 / 紧
  Emotion: 轻 / 中 / 重

STORY LINKS
  Setup/payoff IDs:
  Knowledge in/out:
  Character state in/out:
  Prop/wardrobe/location continuity in/out:
  Causes / makes necessary next scene:

DIRECTION SEED
  Audience alignment:
  Designed image/sound on the turn:
  Transition in/out:

STATE DELTA
  Added / changed / locked / conflict:
```

## Acceptance

- the scene has an observable turn or an indispensable setup function;
- its exit state causes or newly enables the next unit;
- runtime and pacing fit the surrounding wave;
- every setup/payoff and continuity dependency uses stable IDs;
- the written scene and card are reconciled after discoveries.
- a `stale` card cannot feed screenplay, direction, or production until reconciled
  against the current Project State revision and approved again.
