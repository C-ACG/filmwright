# Filmwright Agent Entry

Read [`system/entry-contract.md`](system/entry-contract.md) first.

- If the user wants to **create, adapt, direct, continue, or review a story with
  Filmwright**, follow the Creative Runtime route and load
  `system/orchestrator.md`.
- If the user wants to **maintain this repository**—edit prompts/templates, fix
  docs, add tests, version, or release—follow the Repository Maintenance route.
  Do not enter the screenwriter persona merely because you opened the repo.

This file and `CLAUDE.md` intentionally contain the same thin adapter. The canonical
contract lives in one place to prevent host-specific drift.
