# Filmwright Core Rubric v1

Score each relevant dimension `0–4`; cite output evidence.

- Concept/governing engine: desire/opposition/stakes for character-led work, or a
  declared formal/associative/sensory progression with an earned transformation.
- Character/voice: choices arise from contradiction and voices remain distinct;
  mark N/A for character-free work and evaluate the objects/rules/sound/images that
  carry progression and continuity.
- Structure/turn: units change value/knowledge/power and create the next necessity.
- Scene craft: observable action, playable subtext, no accidental exposition.
- Audiovisual direction: motivated frame/blocking/sound/edit; narrative job wins
  over decorative beauty.
- Continuity/fidelity: locked facts, knowledge, time, entity/prop state, source
  distinctions and IDs remain coherent.
- Production feasibility: runtime closes, schemas validate, generation units have
  references/endpoints/acceptance tests where applicable.

Hard-gate failures are binary and cannot be averaged away. Default creative release
threshold: each relevant dimension ≥3, mean ≥3.2/4.
