# Format · Feature · 长片（75–120 分钟）

> A journey, not an event. A marathon, not a sprint — pacing must breathe. Loads
> `00` + `01` + `02`. Direction modules at stage 7d.

## Feature vs short
- A short is a sprint won on density; a feature needs rises and falls — quiet
  scenes to digest, comic relief to modulate. Sustained 90-min density fatigues.
- A short tells one event; a feature tells a journey — a series of escalating
  trials, each revealing a deeper truth of the character.
- A short's subtext lives in dialogue; a feature's also lives in **scene sequence** —
  a wedding cut against a funeral is subtext made of editing.

## Character depth
Full chain (`00-core-craft.md` §2): `Ghost → Lie → Flaw → Want → trials → Need →
choice → new normal`. Plus a B-story character embodying the opposite of the
thematic argument, and supporting cast each tagged by function (`01-development.md`
§3).

## Structure
Choose a methodology (`02-structure.md`): Save the Cat / Story Circle / McKee /
inner-beat + style variant. Mix freely. Default to Story Circle skeleton if unsure.

## Mandatory elements (default on; flag if missing)
- **Foreshadow & payoff** — 3–5 pairs across layers (prop/line/behavior/structural);
  track with the setup register (`02-structure.md` §6). No unpaid plant.
- **Supporting cast & ensemble** — every speaking role has a function; merge/cut the
  functionless; distinct voices. Ensemble → relationship matrix + POV rhythm.
- **Hidden line 暗线** — 1–2; runs under the surface via clues, surfaces at climax,
  re-colors prior events. (Distinct from a visible subplot.)
- **Subplot weave** — A (outer journey) / B (relationship carrying theme; mirrors A
  internally) / C (optional relief or thematic mirror). A and B cross at the
  midpoint; no subplot becomes a standalone story.
- **Three-layer world** — surface rules / operating logic / underlying truth, each
  revealed through action, never dumped.

## Stage scaling (additions over short)
- **1** add thematic argument (a contestable proposition; A and B take opposite sides).
- **2** synopsis is a one-page treatment (300–500 字; see `templates/treatment.md`).
- **3** add Ghost/Lie/Flaw, B-story character, supporting functions, relationship net.
- **4** fully build: era / protagonist backstory / relationship history / world
  rules (three-layer) / cross-section choice.
- **5** confirm methodology; design and *confirm the opening hook's quality*; mark
  subplot crossings, world-reveal timing, runtime budget, dual-track tags, setup plan.
- **6** group scenes into **sequences** (8–12 for a 90-min film), each with its own
  micro build→conflict→result; runtime + pacing per scene; checkpoint after stage 6.
- **7** plant setups naturally; keep voices distinct; checkpoint each sequence.
- **8** audit A/B/C weave, world-reveal naturalness, "false changes" before the
  Need, opening-hook quality, pacing (runtime sums), setup register, supporting cast.

## Traps
saggy second act (strengthen midpoint, split Act II) · linear arc (need a false
change + retreat) · disconnected B-story (must mirror theme) · climax = "bigger
fight" (must externalize the inner choice) · world dumped at once · plant too
obvious · plant with no payoff · functionless supporting roles · monotone pacing
· no runtime estimate → compressed climax · no breath between heavy scenes ·
tracks always in sync (force some mismatch).
